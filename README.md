# 🥁 Drum Kit

A simple interactive web application that plays drum sounds when users press keys or click buttons.

## 🚀 Features
- Play drum sounds using keyboard keys  
- Clickable drum buttons  
- Visual button animation on interaction  
- Real-time sound feedback  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

## ▶️ Usage
1. Open `index.html` in your browser  
2. Press keys (w, a, s, d, j, k, l) or click buttons  
3. Enjoy the drum sounds  

## 📂 Project Files
- `index.html` – Structure  
- `styles.css` – Styling  
- `index.js` – Functionality
